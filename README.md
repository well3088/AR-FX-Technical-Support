# AR-FX 技术支持（Technical Support）
这是一款使用 AR 技术拍摄特效视频的应用，用户可以通过摄像头在现实场景中叠加特效元素，并录制有趣的视频。
如果您在使用过程中遇到任何问题，请查看以下帮助信息或直接联系我们。
### 如果您在使用过程中遇到任何问题，欢迎通过以下方式与我们联系：
wu.shaopeng@aol.com

### 1. 启动应用后没有出现 AR 特效？
请确保：
- 已授予应用 **相机权限**。
- 使用的是支持 ARKit 的设备（iPhone 8 或更新型号）。
- 有足够光线和清晰的平面可以被检测。

### 2. 为什么视频录制失败？
- 请确认系统剩余存储空间是否足够。
- 确保已授予 **照片库写入权限**。
- 尝试重新启动应用。

### 3. 特效位置不准确？
- 重新移动手机，让 AR 系统重新检测平面。
- 尽量保持光线充足。
- 避免摄像头被遮挡或镜头污渍。

### 4. 应用闪退？
请尝试：
- 重启设备  
- 删除应用并重新安装  
- 更新到最新 iOS 版本  
- 如果问题持续，请通过邮件联系并附上设备型号 + iOS 版本

### 5. AR-FX 非常重视用户隐私。
- 应用不会收集您的任何个人信息。  
- 所有视频都保存在您的设备本地或您选择的 iCloud 照片库中。  
- 应用不会上传任何数据到我们的服务器。

## 🚀 应用功能简介（Features）

- 基于 **ARKit** 的实时特效叠加  
- 支持录制 AR 特效视频  
- 支持保存到照片库
